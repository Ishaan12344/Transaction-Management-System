# File-Based Transaction Management System

## Overview
A Python-based transaction management system that efficiently handles large transaction files using:
- Custom Context Managers
- Generators for lazy processing
- Robust logging and error handling

## Features
- Random transaction generation
- Safe file handling
- Memory-efficient processing
- Transaction summary computation

## Technologies Used
- Python
- File Handling
- Generators
- Logging
- OOP Concepts
